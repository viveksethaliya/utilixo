## Common Causes of Slow Computer

- Too many startup programs
- Low disk space
- Malware or viruses
- Outdated drivers
- Insufficient RAM
- Hard drive fragmentation

## Quick Fixes (5 Minutes)

### 1. Restart Your Computer

Simple but effective! Restart clears temporary files and memory.

### 2. Close Unnecessary Programs

**Step 1:** Press `Ctrl + Shift + Esc` to open Task Manager

**Step 2:** Go to **Processes** tab

**Step 3:** Right-click programs using high CPU/Memory → **End Task**

### 3. Disable Startup Programs

**Step 1:** Open Task Manager (`Ctrl + Shift + Esc`)

**Step 2:** Go to **Startup** tab

**Step 3:** Right-click unnecessary programs → **Disable**

**Step 4:** Restart computer

## Deep Optimization (30 Minutes)

### 1. Free Up Disk Space

**Step 1:** Press `Windows + I` → **System** → **Storage**

**Step 2:** Click **Temporary files**

**Step 3:** Select all categories → **Remove files**

### 2. Run Disk Cleanup

**Step 1:** Search "Disk Cleanup" in Start menu

**Step 2:** Select drive C: → **OK**

**Step 3:** Check all boxes → **Clean up system files**

### 3. Scan for Malware

**Step 1:** Open Windows Security

**Step 2:** Click **Virus & threat protection**

**Step 3:** Run **Quick scan**

## Advanced Solutions

### Upgrade to SSD

- Fastest way to improve performance
- 10x faster than traditional hard drives
- Clone your current drive

### Add More RAM

- Minimum 8GB recommended
- 16GB for gaming/heavy use
- Easy DIY upgrade

### Reinstall Windows

Last resort if nothing else works. Backs up important files first!
