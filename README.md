# Utilixo - Next.js Version

Complete migration from Vite to Next.js with SSG for perfect SEO.

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Build for Production

```bash
npm run build
```

Static files will be in `/out` folder.

## Deploy to Vercel

```bash
# Push to GitHub
git init
git add .
git commit -m "Initial commit"
git push

# Then import in Vercel dashboard
```

## Project Structure

- `app/` - All pages and routes
- `app/tools/` - Interactive tools (client components)
- `app/fixit/` - Fix-it guides (SSG)
- `content/fixit/` - Markdown articles
- `lib/` - Utilities
